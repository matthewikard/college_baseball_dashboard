## College Baseball Dashboard

College Baseball coverage is lacking. Up until a couple years ago you couldn't even check scores on ESPN!
Thankfully ESPN does offer college baseball scores now, but it's still lacking common datapoints I'm looking to find.

This dashboard uses the unofficial ESPN API to display SEC baseball scores with a few additional features:
* Series Record
* Overall Record and Conference Record
* Standing in SEC
* W/L streak

The dashboard is hosted on the free tier of Flask [here](https://college-baseball-dashboard-n0lo.onrender.com), so an initial load will take about 15-30 seconds.
